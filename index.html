A machine learning algorithm for estimating the probability that a given mutation profile corresponds to a particular tumor type

RESEARCH USE ONLY

1. Enter comma-delimited list of capitalized genes reported as mutant in a clinical cancer sequencing report.
e.g. (EGFR, KRAS, TP53)
2. Click submit and wait patiently(!). It takes around 30 seconds to build the classifiers and report the results.
